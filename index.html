Anti-Harvest Script

Place this Javascript in your HEAD tag
<script type="text/javascript">

  var name = 'jdm113'; 
  var server = 'zips.uakron';
  var suffix = '.edu'; 
  function noHarvest() 
  { 
    document.write('<a href="' + 'mailto:' + name + '@' + server + suffix + '">' + 'jdm113 |at| zips.uakron |DOT| edu' +'</a>'); 
  }

</script>
place in BODY where you want the email link:
<script> noHarvest() </script>
